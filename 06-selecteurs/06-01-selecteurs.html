1- Ce titre (#titreNiveau2) sera sélectionné dans la console

Dans la console nous voyons les différentes méthodes de sélectionner cet élément dont l'Id est titreNiveau1

  1. Dans une variable qui se trouve dans window.onLoad() /// var select1 = titreNiveau1
  2. Dans une variable simple var select2 = titreNiveau1;
  3. Avec getElementById directement dans le document document.getElementById('titreNiveau1');
  4. Avec getElementsByClassName je récupère un ensemble une collection de class getElementsByClassName('maClasse1');
  5. Avec getElementsByTagName je récupère un ensemble d'élément par leur nom
