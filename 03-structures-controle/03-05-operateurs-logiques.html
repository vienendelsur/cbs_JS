JS - III 05 Les opérateurs logiques

Ce type d'opérateur va nous permettre d'effectuer plusieurs comparaisons dans nos tests ou d'inverser la valeur logique du résultat d'un test.

1- Présentation et utilisation des opérateurs logiques

Le JS supporte 3 opérateurs logiques : l'opérateur logique "ET", "OU" et "NON".

Opérateurs logiques
Opérateurs, nom Opérateur, symbole Description
AND (ET) &&

renvoie true si toutes les comparaisons sont évaluées à true, ou false sinon
OR (OU) || renvoie true, si au moins l’une des comparaisons est évaluée à true, ou false sinon
NO (NON) ! Renvoie false si une comparaison est évaluée à true, ou renvoie true dans le cas contraire
